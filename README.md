# Unlocking Societal Trends in Aadhaar Enrolment and Update Ecosystem

## Overview
This project analyses UIDAI Aadhaar Enrolment, Demographic Update, and
combined governance datasets to identify:

- Coverage saturation
- Lifecycle transition pressure
- Update demand forecasting
- Governance risk zones
- Regional inequality

## Datasets Used
- Aadhaar Enrolment Dataset
- Aadhaar Demographic Update Dataset

(All datasets are aggregated and anonymised as provided by UIDAI.)

## Methodology
- Data cleaning & validation
- State and district standardisation
- Feature engineering
- Univariate, bivariate & trivariate EDA
- Advanced governance indicators
- Decision matrix & policy framework

## Key Outputs
- State–month analytical datasets
- Governance risk score
- Policy action classification
- Storytelling dashboards
